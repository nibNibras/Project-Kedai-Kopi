# ☕ kedaiKopiku: Coffee Store Landing Page

A premium, fully responsive landing page designed for a modern coffee shop experience. Built from scratch with native web technologies, this platform showcases the store's finest brews, brand philosophy, and features an integrated product menu and dynamic customer contact system.

# 🌐 Website Sections
The landing page features four beautifully integrated sections:

- <strong>🏠 Home (Hero):</strong> Welcomes visitors with a striking hero banner ("Mari Nikmati Secangkir Kopi"), a rich coffee-themed background, and an interactive "Beli Sekarang" call-to-action button.

- <strong>✨ About Us:</strong> A dedicated narrative space outlining the heritage of our beans, our roasting process, and the core philosophy behind brewing the perfect cup.

- <strong>☕ Menu & Products:</strong> A clean grid layout showcasing our premium beverage options (e.g., Espresso starting at IDR 15K). Each item display includes a high-quality product preview card and interactive pricing.

- <strong>📞 Contact Us:</strong> A minimalist, user-friendly feedback form equipped with fields for Name, Email, and Phone Number, allowing customers to easily send inquiries or book catering.
